JavaScript - why It's cool

JavaScript is one of the most popular programming language to learn in 2020, if not the most!

In order to learn JavaScript you need to understand what is a programming language first.
Programming language is the way we communicate with the computer or rather it's the way we tell the computer what to do.

The way programming languages work is, you write code and the compiler transforms the code into assembly language which then transforms into binary, which is the actual language computer understands.
Binary is a series of 0's and 1's that represents electrical currency flow, 1 means close circuit (electric flow) and 0 means open circuit (no electric flow).
Binary is very diffucult and unefficient to write and thet's why we need programming languages.

JavaScript is a high level language, which means the built in commands and functions are easier to understand than lower level programming languages since the higher the level of the language is, the closer it is to english.

JavaScript is mostly used in Web development, it's a client side programming language (even though it has frameworks which you can use in the server side/backend of you website), that means JavaScript is not communicating with your computer but with your web browser.

After we understood WTH JavaScript is, we can start learning it.

JavaScript applications are built using Variables and Functions.

Let's start:

Data Types:

  • String - represents sequence of characters e.g. "hello"
  • Number -represents numeric values e.g. 100 (can also represent decimal values e.g 100.123)
  • Boolean - represents boolean value either false or true
  • Undefined - represents undefined value
  • Null - represents null i.e. no value at all


Variables:

To declare a variable we need to use the built in command: "var", which represents variable.
Here's an example:

var x = 5;
var y = 6;
var z = x + y; (which means z = 5 + 6 / z = 11)
var person = "Avinoam Ruach";


What I like about JavaScript is that we don't need to specify what data type our variable is, JavaScript knows automatically what data type the variable is based on the value of that variable.

Loops:

Let's say you want to print to the console the numbers from 1-10.
You can use the built in command: "console.log('Input here what you want to display in the console')", but to display the numbers from 1-10 you will need to type that command 10 times, which may not seems a lot but what if you want to print to the console the numbers between 1-1,000?
Thats 1000 lines! it will be much more time consuming and much less efficient. That's why we use loops.
Here's an example:

for (var i = 0; i < 1000; i++ {
console.log (i);
}

This loop prints the numbers 1-1000.

  • While loop or Do/While loop

Functions:

Functions are blocks of code that you can assign name to. Unlike variables functions can contain more than 1 data type in them and can contain loops, algorithms, variables, etc..
Let's see an example:

function sum(num1, num2) {
return num1 + num2;
}

This function is used to calculate the sum of 2 parameters (num1 and num2).
Parameters are the values we insert into the parenthesis, in our case (num1 and num2).
When we use more than 1 parameter we will use comma "," to saparate the values and indicate each one of the values as a parameter.
Parameters are like unassigned variable, you give the variable a name but not a value (yet).
Just like the next example of unassigned variable:
var noValue;

We have a variable named "noValue", and we didn't gave it a value.
In order to use the parameters we need to define them when we call the function to work, Here's an example for that:

sum (10,20) // Will result 30 because we assigned num1 to the value 10, and num2 to the value 20.

// We can assign not only raw values but variables as well, let's first create some variables to work with:

var firstNumber = 5;
var secondNumber = 6;
// we will now call the function using those 2 variables
sum (firstNumber, secondNumber) // The sum will be 11 (5 + 6).
